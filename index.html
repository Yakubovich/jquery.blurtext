Navigation Example

To specify both blur amount and duration, pass them as an object.

$("li").blurText(10);
$("li").hover(function(){
  $(this).blurText("to", { blur: 0, duration: 500});
}, function() {
  $(this).blurText("to", { blur: 10, duration: 500});
});
